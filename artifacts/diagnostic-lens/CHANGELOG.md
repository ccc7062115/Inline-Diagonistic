# Changelog

## [0.1.0] - 2026-04-19

### Added

- Initial release
- Inline pill indicators for errors, warnings, hints, and info
- Dot mode (cursor elsewhere) and message mode (cursor on diagnostic line)
- Configurable transparency blending and opacity
- Configurable per-severity toggles
- Configurable max message length
- Debounced updates for performance
- Decoration type caching to minimize resource usage
- Contrast-aware pill text color
